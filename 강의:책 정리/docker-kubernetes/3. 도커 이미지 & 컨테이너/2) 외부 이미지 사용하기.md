
이미지를 생성하고 가져오는 방법 2가지
- 이미 존재하는 이미지 사용하기 (공식 이미지 등) 
	- `DockerHub`에서 사용 가능
- 커스텀 이미지 사용하기

### 컨테이너로 실행하기
run 커맨드를 사용해서 인스턴스를 생성한다.
```docker
docker run node -- node를 실행한다 (컨테이너가 바로 종료되며, 실행에 큰 의미가 없음)
docker run -it node -- 컨테이너와 쉘이 상호작용한다.
```
